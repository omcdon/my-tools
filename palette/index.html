How to Use

  • Upload any image (photo, screenshot, artwork, mood board).
  • The tool analyzes pixel colors and extracts the 8 most dominant ones.
  • Click any color to copy its HEX code. Use "Copy All" for the full palette.
  • "Export as CSS" generates CSS custom properties you can paste into your project.

Use Cases

  • Extract brand colors from a logo or style guide
  • Create color schemes from mood boards or reference images
  • Match text/overlay colors to background photos
  • Build cohesive palettes for web design projects

Frequently Asked Questions

  • What image formats are supported? — You can upload PNG, JPG, WebP, and GIF images. The tool analyzes the image pixels to extract the most dominant colors.
  • How many colors does it extract? — By default, it extracts the top 5-8 dominant colors. You can adjust the number of colors in the settings.
  • Can I copy the color codes? — Yes! Click on any color swatch to copy its HEX code to your clipboard. The tool also shows RGB and HSL values.
