Frequently Asked Questions

  • What is a User Agent? — A User Agent (UA) string is text your browser sends to websites identifying itself, including browser name, version, and operating system.
  • Is any data sent to a server? — No. All detection runs locally in your browser using JavaScript APIs. Nothing is transmitted.
  • Why does my GPU show as different? — The GPU info comes from WebGL, which reports the graphics driver's renderer string. It may differ from your actual hardware model name.
  • Why is some data showing as "Unavailable"? — Some APIs (like Device Memory or Network Information) are not supported in all browsers. Chrome supports the most features.
  • Can websites detect my real location? — This tool only shows what your browser reports. Websites can request geolocation permission separately, which you can deny.
