Frequently Asked Questions 常见问题

  • What are the error correction levels? — L (Low) = ~7% recovery, M (Medium) = ~15%, Q (Quartile) = ~25%, H (High) = ~30%. Higher levels allow the QR code to be read even if partially damaged, but increase the code density.
  • How much data can a QR code hold? — A standard QR code can hold up to 4,296 alphanumeric characters or 7,089 numeric characters. With higher error correction, the capacity decreases.
  • Will the QR code expire? — No. A QR code is simply a visual encoding of data — it does not expire. However, if the QR code encodes a URL, the website at that URL may change or go offline.
  • 纠错等级是什么意思？ — L（低）≈ 7% 恢复能力，M（中）≈ 15%，Q（四分之一）≈ 25%，H（高）≈ 30%。等级越高，即使二维码部分损坏也能读取，但代码密度会增加。
  • 二维码能存多少数据？ — 标准二维码最多可存储 4,296 个字母数字字符或 7,089 个数字字符。纠错等级越高，容量越小。
  • 二维码会过期吗？ — 不会。二维码只是数据的可视化编码——它本身不会过期。但如果二维码编码的是 URL，该网站可能会更改或下线。
